# account

twnyan で使用するアカウントの管理・切り替えを行います。

```
twnyan account <コマンド>
twnyan acc <コマンド>
```

## コマンド

### account add

アカウントを twnyan に追加します。

```
twnyan account add
```

### account list

twnyan に追加されているアカウントを一覧表示します。

```
twnyan account list
twnyan account ls
```

### account remove

twnyan からアカウントを削除します。

```
twnyan account remove <ユーザ名>
twnyan account rm <ユーザ名>
```

### account switch

> 対話モードでのみ使用可能

使用するアカウントを切り替えます。

```
twnyan account switch <ユーザ名>
twnyan account sw <ユーザ名>
```
