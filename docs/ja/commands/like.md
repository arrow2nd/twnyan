# like

ツイートにいいねします。

```
twnyan like {<ツイート番号>... | <コマンド>}
twnyan lk {<ツイート番号>... | <コマンド>}
twnyan fv {<ツイート番号>... | <コマンド>}
```

- ツイート番号は、半角スペースで区切ることで複数指定できます

## コマンド

### like remove

ツイートのいいねを取り消します。

```
twnyan like remove <ツイート番号>...
twnyan like rm <ツイート番号>...
```

- ツイート番号は、半角スペースで区切ることで複数指定できます

### like help

ヘルプを表示します。

```
twnyan like help
```
