# block

ユーザをブロックします。

```
twnyan block {<ユーザ名 / ツイート番号> | <コマンド>}
twnyan bk {<ユーザ名 / ツイート番号> | <コマンド>}
```

## コマンド

### block remove

ユーザのブロックを解除します。

```
twnyan block remove <ユーザ名 / ツイート番号>
twnyan block rm <ユーザ名 / ツイート番号>
```

- ユーザ名の `@` は省略可能です

### block help

ヘルプを表示します。

```
twnyan block help
```
