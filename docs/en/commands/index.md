# twnyan

twnyan is a simple Twitter client that runs in the terminal.

```
twnyan [Options] [Command]
```

## Options

- `-A` `--account <User Name>`
  - Specify account to use (if not specified, the main account will be used)
- `-H` `--help`
  - Display help with options
