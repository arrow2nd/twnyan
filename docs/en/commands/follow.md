# follow

Follow a user.

```
twnyan follow {<username / tweet-number> | <command>}
twnyan fw {<username / tweet-number> | <command>}
```

- The `@` in the user name can be omitted

## Command

### follow remove

Unfollow a user.

```
twnyan follow remove <username / tweet-number>
twnyan follow rm <username / tweet-number>
```

- The `@` in the user name can be omitted

### follow help

Displays help.

```
twnyan follow help
```
