# block

Blocks the user.

```
twnyan block {<User Name / Tweet Number> | <Command>}
twnyan bk {<User Name / Tweet Number> | <Command>}
```

- The `@` in the user name can be omitted

## Command

### block remove

Unblock a user.

```
twnyan block remove <User Name / Tweet Number>
twnyan block rm <User Name / Tweet Number>
```

- The `@` in the user name can be omitted

### block help

Displays help.

```
twnyan block help
```
