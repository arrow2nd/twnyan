# mute

Mutes the user.

```
twnyan mute {<User Name / Tweet Number> | <Command>}
twnyan mu {<User Name / Tweet Number> | <Command>}
```

## Command

### mute remove

Unmutes the user.

```
twnyan mute remove <User Name / Tweet Number>
twnyan mute rm <User Name / Tweet Number>
```

- The `@` in the user name can be omitted

### mute help

Displays help.

```
twnyan mute help
```
