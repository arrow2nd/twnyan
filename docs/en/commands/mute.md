# mute

Mute a user.

```
twnyan mute {<username / tweet-number> | <command>}
twnyan mu {<username / tweet-number> | <command>}
```

- The `@` in the user name can be omitted

## Command

### mute remove

Unmute a user.

```
twnyan mute remove <username / tweet-number>
twnyan mute rm <username / tweet-number>
```

- The `@` in the user name can be omitted

### mute help

Displays help.

```
twnyan mute help
```
