# likert

Like & Retweet tweets.

```
twnyan likert {<Tweet Number>... | <Command>}
twnyan lr {<Tweet Number>... | <Command>}
twnyan fr {<Tweet Number>... | <Command>}
```

- Multiple tweet numbers can be specified by separating them with a single space

## Command

### likert help

Displays help.

```
twnyan likert help
```
